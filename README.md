# quran-hd-ayah-coordinates

A structured dataset of high-resolution Quran page images with precise ayah-level coordinate mappings for highlighting and interaction.

---

## 📦 What’s Included

- 📖 549 Quran pages as HD images  
- 📍 Per-page JSON files with bounding box coordinates for each ayah  
- 🧭 Coordinates aligned for accurate highlighting, tapping, and overlays  

Each JSON file contains the exact position of every ayah on that page using pixel-based coordinates.
